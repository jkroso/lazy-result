
# lazy-result

  lazy evaluation using [Results](//github.com/jkroso/result)

## Getting Started

_With [component](//github.com/component/component), [packin](//github.com/jkroso/packin) or [npm](//github.com/isaacs/npm)_  

	$ {package mananger} install jkroso/lazy-result

then in your app:

```js
var lazy-result = require('lazy-result')
```

## Running the tests

Just run `make`. It will install and start a development server so all you then need to do is point your browser to `localhost:3000/test`. Likewise to run the examples.
